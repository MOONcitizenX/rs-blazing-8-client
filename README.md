## Run Locally

Clone the project

```bash
  git clone https://github.com/MOONcitizenX/rs-blazing-8-client.git
```

Go to the project directory

```bash
  cd rs-blazing-8-client
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
